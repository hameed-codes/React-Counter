# React Counter

> A clean and animated counter app built with React; my first hands-on project while learning React fundamentals like useState and CSS Modules.

**Live Demo:** [Click here]()

![screenshot]()

---

## Features

- Increment, decrement, and reset the counter
- Smooth scale-pop animation on every button click
- Scoped component styling using CSS Modules
- Clean and responsive UI

---

## Built With

- React 18
- Vite
- CSS Modules
- JavaScript (ES6+)

---

## Getting Started

### Prerequisites

- Node.js v18+
- npm

### Installation

```bash
git clone https://github.com/hameed-codes/react-counter.git
cd react-counter
npm install
npm run dev
```
